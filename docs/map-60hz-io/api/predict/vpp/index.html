집합발전소 발전량예측 #

집합발전소의 예측발전량 값을 API로 받을 수 있습니다.

URL 준비 #

알림에서 xml 사용 설정을 누릅니다. 그러면 해당 집합 발전소의 전용 API URL이 생성됩니다.

API 명세 #

  • URL

    /vpp/share

  • Method:

    GET

  • URL Params

    변수명 타입 필수 설명
    token str O token
    type str X 응답값의 형태(‘xml’ or ‘json’). default는 xml
    dt int X YYYYMMDD의 숫자, default는 내일날짜

  • Success Response:

    • Code: 200

    • 출력값

      출력명 타입 설명
      name str 집합발전소 이름 or 발전소 이름
      unit str 단위
      predict array 예측발전량 어레이
      datetime str 시간
      generation float 예측발전량
      plants array 발전소 어레이
      id int 발전소 ID
      plant_name str 발전소 이름

      Content:

      <?xml version="1.0" encoding="UTF-8" ?>
<root>
    <name>{집합발전소 이름}</name>
    <unit>kWh</unit>
    <update_datetime>2021-06-09 07:00:00+09:00</update_datetime>
    <predict>
        <item>
            <datetime>2021-06-10 00:00:00+09:00</datetime>
            <generation>0.0</generation>
        </item>
        <item>
            <datetime>2021-06-10 01:00:00+09:00</datetime>
            <generation>0.0</generation>
        </item>
        ...
        <item>
            <datetime>2021-06-10 20:00:00+09:00</datetime>
            <generation>2.54</generation>
        </item>
        <item>
            <datetime>2021-06-10 21:00:00+09:00</datetime>
            <generation>0.0</generation>
        </item>
        <item>
            <datetime>2021-06-10 22:00:00+09:00</datetime>
            <generation>0.0</generation>
        </item>
        <item>
            <datetime>2021-06-10 23:00:00+09:00</datetime>
            <generation>0.0</generation>
        </item>
    </predict>
    <plants>
        <item>
            <id>{발전소 ID}</id>
            <plant_name>{발전소 이름}</plant_name>
        </item>
        <item>
            <id>{발전소 ID}</id>
            <plant_name>{발전소 이름}</plant_name>
        </item>
    </plants>
</root>

  • Error Response:

    • Code: 400 BAD REQUEST

      Content: { message : "잘못된 토큰입니다." }

      Content: { message : "잘못된 요청입니다. (dt는 YYYYMMDD 포맷이여야 합니다.)" }

      Content: { message : "잘못된 요청입니다. (dt는 현재날짜기준 이틀뒤까지 요청가능합니다.)" }

  • Sample Call:

    Bash # 

    $ curl https://{API_URL}/vpp/share?token={token}

    Python3 # 

    >>> import requests
>>> URL = "https://{API_URL}/vpp/share?token={token}"
>>> rst = requests.get(URL)
>>> rst.status_code
200
