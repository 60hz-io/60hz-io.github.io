세계 기상정보 확인 #

개요 #

세계 기상정보를 햇빛지도에서 확인할 수 있습니다. 세계 기상정보는 미국 해양대기청에서 제공하는 Global Forecast System 데이터를 사용하고 있으며, 햇빛지도에서는 오늘부터 내일, 내일 모레까지의 데이터를 확인할 수 있습니다.

풍속, 풍향 #

왼쪽의 바람 버튼을 누르게 되면 지도에 전 세계의 풍속(m/s)과 풍향 정보가 지도에 표시됩니다.
