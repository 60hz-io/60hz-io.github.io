집합 이름 변경 #

집합 이름을 변경하기 위해 “집합1"이라는 텍스트를 클릭합니다. 그러면 이름을 수정할 수 있도록, 커서가 표시됩니다.

바꾸고자 하는 이름으로 수정한 뒤, 다른 곳을 클릭하였을 때 자동적으로 이름이 수정 완료됩니다.
