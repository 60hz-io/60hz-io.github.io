집합에 발전소 추가 #

발전소 검색 #

하위 집합 옆의 추가 버튼을 클릭합니다.

발전소 검색창에서, 검색하고자 하는 발전소의 이름 혹은 발전소 이름의 포함된 단어를 입력하고 검색을 합니다.

그러면 위와 같이 발전소 이름에 단어가 포함된 발전소 목록이 나옵니다. 검색 결과는 최대 100건까지만 조회됩니다.

사용자가 신규생성한 발전소도 검색을 할 수 있습니다. 신규 생성한 발전소의 이름을 기입하고 검색합니다.

그러면 위와 같이 신규생성한 발전소가 검색 목록에 조회됩니다.

집합에 발전소 추가 #

발전소 검색 후, 집합에 추가하고자 하는 발전소의 체크박스를 클릭한 뒤 추가 버튼을 클릭합니다.

그러면 해당 집합의 추가된 발전소 목록과 용량, 발전소 갯수가 업데이트 됩니다.
