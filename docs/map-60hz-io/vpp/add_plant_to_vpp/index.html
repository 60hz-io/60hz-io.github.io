집합에 발전소 추가 #

발전소 검색 #

하위 집합의 [추가] 버튼을 선택합니다.

검색하고자 하는 발전소의 이름이나 이름에 포함된 단어를 입력하고 [검색]을 선택합니다.

검색 결과는 최대 100건까지 표시됩니다.

신규생성을 통해 추가시킨 발전소도 검색이 됩니다.

집합에 발전소 추가 #

검색된 발전소 좌측의 체크박스를 선택하고 [추가] 버튼을 선택합니다.

발전소 집합에 발전소가 추가 되었습니다.
