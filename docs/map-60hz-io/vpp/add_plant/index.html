발전소 신규생성 #

식스티헤르츠가 사전에 등록시켜 놓은 발전소가 아닌 경우 새로운 발전소를 추가시킬 수 있습니다.

이렇게 등록된 발전소는 등록시킨 사용자에게만 노출 됩니다.

발전소 신규 생성 메뉴 열기 #

하위 집합의 [추가] 버튼을 선택합니다.

좌측하단의 [신규생성] 버튼을 선택합니다.

신규 생성을 위한 입력창이 열립니다.

발전소 신규 생성하기 #

발전소의 종류를 선택하고 이름, 주소, 용량을 입력합니다. 방위각은 정북일 경우 0도, 정남일 경우 180도 입니다. 정보를 모두 입력하였으면 [저장]을 선택합니다. 앞으로 다양한 설정 메뉴가 지속적으로 추가될 예정입니다.
