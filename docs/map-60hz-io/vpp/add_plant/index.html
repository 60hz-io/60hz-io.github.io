발전소 신규생성 #

집합자원에 발전소 구성 시 신규로 발전소를 등록 및 추가를 하고싶을 때는 발전소를 신규생성을 할 수 있습니다.

발전소 신규 생성 메뉴 열기 #

하위 집합의 추가 버튼을 클릭합니다.

좌측하단의 신규생성 버튼을 클릭합니다.

그러면 발전소 신규생성창이 열리게 됩니다.

발전소 신규 생성하기 #

등록할 발전소의 종류에 따라 태양광, 풍력을 클릭합니다. 발전소의 이름, 주소, 용량(kW) 등을 기입합니다. 방위각은 정북일 경우 0도, 정남일 경우 180도 입니다. 정보를 모두 기입하였으면 저장을 클릭합니다.
