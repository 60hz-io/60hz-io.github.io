집합 추가 #

기본 구성 #

집합을 추가하기 위해 집합구성 메뉴를 누르면 위와 같은 화면이 나옵니다.

처음으로 집합구성을 사용하는 사용자는 기본적으로 하나의 집합이 구성되어 있습니다.

하위 집합 추가 #

하위 집합을 추가하기 위해 “추가” 버튼을 누릅니다. 추가 버튼을 누른 횟수 만큼 하위 집합이 추가됩니다.

하위 집합 삭제 #

하위 집합을 삭제하기 위해 “삭제” 버튼을 누릅니다. 그러면 “집합발전소 집합을 삭제하시겠습니까?” 라는 알람창이 뜨게 되고, 확인을 누르면 하위 집합이 삭제가 됩니다.
