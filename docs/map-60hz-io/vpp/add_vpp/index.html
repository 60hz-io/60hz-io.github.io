집합 추가 #

기본 구성 #

상단의 메뉴에서 [집합구성]을 선택합니다. 처음 시작할 때는 “집합1"이라는 이름의 기본 집합이 존재합니다.

하위 집합 추가 #

하위 집합을 추가시켜 계층구조를 만들 수 있습니다. [추가] 버튼을 선택합니다

하위 집합 삭제 #

[삭제] 버튼을 선택하면 하위 집합을 삭제할 수 있습니다.
